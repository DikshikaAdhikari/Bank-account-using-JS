🏦 Bank Account System (JavaScript)

A simple Bank Account Management System built using JavaScript. This project demonstrates basic object-oriented programming (OOP) concepts such as classes, objects, methods, and encapsulation.

🚀 Features

Create a bank account

Deposit money

Withdraw money

Check balance

Simple transaction validation (no overdraft)

Console-based interaction (can be extended to UI)

🛠️ Technologies Used

.JavaScript (ES6+)

.Node.js (optional, if running outside browser)

.Console/Terminal
